# **Automation Summary**

### Quick Summary of Automation Types

- **Build Automation:**  
  Tools: Maven, Gradle, Jenkins, Ant  
  Example: Jenkins CI/CD for Java app
- **Test Automation:**  
  Tools: Selenium, JUnit, TestNG, Cypress  
  Example: Selenium for e-commerce testing
- **Deployment Automation:**  
  Tools: Jenkins, Ansible, Docker, Kubernetes  
  Example: Docker & Kubernetes for microservices
